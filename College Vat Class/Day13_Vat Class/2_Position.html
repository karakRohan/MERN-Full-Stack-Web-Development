Position Properties

There are five different position values:-

  1. Static
  2. Relative: Position The Element Relative to Its Normal Postion And will leave a gap at its normal position.
  3. Fixed: Position The element relative to the browser window.
  4. Abolute: Position The element relative to the position of its first parent.
  5. Sticky: Position The element relative to the user scroll position.
